## "Удалить запятые"

### Задание

Удалите из строки все запятые.

### Формат входных данных

Дана произвольная строка.

### Формат выходных данных

Вывести исходную строку без запятых.

### Решение задачи

```python
input_str = input("input_str: ")
output_str = input_str.replace(",", "")
print(output_str)

```
